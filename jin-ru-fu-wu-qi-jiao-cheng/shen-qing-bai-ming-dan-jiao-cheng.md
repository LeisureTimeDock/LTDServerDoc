# ✈️ 申请白名单教程

{% hint style="warning" %}
本教程可能会过时，详细以服主公布为准。
{% endhint %}

首先我们是一个正版白名单制的服务器，你得有一个Minecraft Java版账户才能申请白名单。

如何购买正版，这边就不多赘述了。

<details>

<summary>如何购买Minecraft正版</summary>

别买什么MC账户黑卡什么的，MC目前入正的花费不算高。

你可以在视频网站搜&#x7D22;**“Minecraft购买指南”** ，建议去看发布时间最近且播放量可以的，

这里推荐一个B站上搜索的教程。

[Minecraft购买指南。告别锁区](https://www.bilibili.com/video/BV13qBsBUEbw)

</details>

当你有了正版时，可以登录我们白名单系统：

{% embed url="https://whitelist.r3944realms.top/" %}

<figure><img src="../.gitbook/assets/image (2).png" alt="" width="563"><figcaption></figcaption></figure>

输入QQ号和正版游戏ID即可进入白名单申请界面。

<figure><img src="../.gitbook/assets/image (3).png" alt=""><figcaption></figcaption></figure>

按步骤操作后点击打开白名单问卷按钮。答完题后需要验证邮箱。

{% hint style="danger" %}
超过时间未验证，再次登录则会被清除掉答题数据，意味着你要重新答了。
{% endhint %}

<figure><img src="../.gitbook/assets/image (4).png" alt="" width="311"><figcaption></figcaption></figure>

检测你的QQ邮箱，是否有相关邮件。

如收件箱果没有，检查一下垃圾箱，实在确认没有发送可以联系服主检查下。

<figure><img src="../.gitbook/assets/image (6).png" alt=""><figcaption></figcaption></figure>

请在规定时间里验证，未验证的申请请求将不会通过。
